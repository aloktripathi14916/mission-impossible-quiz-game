# 🎯 Mission Impossible Quiz Game

An interactive browser-based trivia survival game built using HTML, CSS, and JavaScript with dynamic API-fetched questions, countdown gameplay, selectable agents, and persistent mission logs.

---

## 🚀 Live Demo

🔗 https://mission-impossible-quiz-game.netlify.app/

---

## ✨ Features

- 🎮 Interactive mission-based trivia gameplay
- ⏳ Countdown timer system
- ❤️ Lives and score tracking
- 🌐 Questions fetched dynamically from Open Trivia DB API
- 📜 Match history stored using localStorage
- 🕵️ Selectable playable agents with unique abilities
- 🔁 Restart and replay system
- 💻 Responsive futuristic UI design

---

## 🕹️ Playable Characters

### 🟢 ECHO
> New recruit with no special abilities.

### 🟡 ORACLE
> Removes one incorrect option from every mission.

### 🔵 GLITCH
> Gains +2 extra seconds for each mission.

### 🔴 REAPER
> Starts the game with one extra life.
